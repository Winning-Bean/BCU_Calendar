## Offer service
- 사용자의 공개/비공개를 통한 일정/사진 구분 이용 서비스 제공
- 사용자의 친구와 단순 일정/사진 공개를 통한 공유 서비스 제공
- 다수 사용자들의 그룹에서의 일정/할 일/사진 공유 서비스 제공
